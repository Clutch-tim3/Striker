About

Mahoraga is a self-learning endpoint defence system that detects, classifies, and neutralises threats in real time — without signature updates or cloud subscriptions. It runs entirely on your device, monitors six sensor layers simultaneously, and builds its own antibody library from every threat it encounters.

Unlike traditional antivirus that relies on known-bad databases, Mahoraga uses an Isolation Forest anomaly model and a behavioural classifier trained on your own machine's telemetry. Every threat it defeats makes it smarter — and through federated learning, insights from the global fleet improve every installation without sharing your data.

  • Process monitoring with platform-specific detection rules for macOS, Windows, and Linux
  • File system watcher covering persistence paths (LaunchAgents, Startup folder, cron.d)
  • Network traffic analysis for C2 beaconing, data exfiltration, and suspicious ports
  • Memory scanner detecting W^X violations and shellcode injection patterns
  • Machine learning anomaly detection — no signature database needed
  • MITRE ATT&CK framework mapping for every detected threat
  • Antibody archive that stores and replays every neutralisation
  • Federated learning — your model improves globally while your data stays local
  • Threat map visualising attack origins in real time on a 3D globe
